# FaceLock Private Message 🔒

This is a full-stack web app where users can create and share private messages securely.

## Features
- User Authentication (Login/Register)
- Create Secret Messages
- Share via Unique Link
- View Messages securely

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express

## Run Locally

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
